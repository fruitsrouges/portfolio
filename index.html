KODATA

HTML / CSS / JS

KODATA est un site web dédié à la data visualisation de données sur les films d’animations Ghibli. Nous avons récupéré des jeux de données provenant de Wikidata, d'une API qui recensent tous les films Ghibli, ainsi que de l'API de Spotify pour les bandes sonores des films. Puis, nous avons conçu et intégré des graphiques interactifs pour visualiser ces données à l'aide de Flourish et de JavaScript.

ESOLY

Direction artistique

Le documentaire Et si on levait les yeux ? met en lumière l’omniprésence des écrans et leur impact sur les enfants, aggravé par le manque d’encadrement parental, ce qui nous a conduits à créer ESOLY, une application ludique prenant la forme d’un jeu de simulation. Son objectif est de limiter l’usage excessif des écrans tout en renforçant les liens familiaux. Elle aide les enfants à s’auto-réguler, avec une implication active des parents dans cette démarche.
